The code included in this submission contains the following:

- "Team13 - 1stSemester.zip"
  - 1st semester code, single threaded, all main execution in main.c with calls to supporting libraries included.
  - Makes use of the Standard Peripheral Library supplied by ST Microelectronics

- "Team13 - 2ndSemester.zip"
  - 2nd semester code, multi-threaded, threads and queues are created in main.c as well as the kernel is fired in main()
  - Thread source is named "*_task.c"
  - Makes use of FreeRTOS V8.2.1, CMSIS RTOS API, CMSIS Hardware Abstraction Layer and originates from STM32CubeMX using their libraries
